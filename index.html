<h2>How it started</h2>

<p>Hey y'all! I'm Mikaela, and I love learning how to make things pretty. My background in design started with graphics, then I veered towards illustration, then I was asked to build a website, and my ambitious self thought, "Sure! I'll just do it from scratch because I hate the overuse of website builder templates"</p>

<p>I logged into my favorite trusty resource: Reddit, found some communities that seemed related to what I was trying to accomplish, scoured different threads about the best way to build a completely customized website, and after initially trying the easy way out with Webflow and *sigh* Elementor, I realized that maybe just maybe I'm going to have to learn a thing or two about HTML, CSS, and JavaScript to implement the lofty ideas I had for this project.</p>

<p>Fun fact: there are many spirited Reddit debates about the best resources to learn web development. I started with freeCodeCamp but then I realized as I was flying through the modules I wasn't really hanging on to the concepts. So I went back to Reddit to find another way, and that's when I learned about The Odin Project. I liked The Odin Project because of its thoroughness, and I liked freeCodeCamp because I was able to implement the concepts I was learning immediately.</p>

<p>So I'm doing both, and that's where I am today. I created this site to finish my Web Design Certification on freeCodeCamp then I'll jump right into JavaScript. I can't wait to make my sites more functional and have all the cool transitions I envision in my head.</p>

<h2>How it's going</h2>

<p>When I first started reading about how to learn web development, the one piece of advise I heard constantly was to do projects. You can read all the videos in the world and watch all of the tutorials on YouTube, but if you're not creating something while you're learning, chances are, you aren't learning as efficiently as possible.</p>

<h3>Google Homepage</h3>

<p>Here is the first project I created and published from GitHub. I recreated the google homepage for an assingment from The Odin Project. This project really helped me understand the box model and how to align elements horizontally. Flex became my best friend because of this assignment. And I became quite familiar with the Inspection tool in Google Chrome.</p>
